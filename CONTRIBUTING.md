# Contribuyendo a Privé Chat

¡Gracias por tu interés en contribuir a Privé Chat! Queremos que el proceso sea lo más fácil y transparente posible.

## Proceso de Pull Request

1. Asegúrate de que las dependencias estén instaladas (`npm install`).
2. Crea una rama para tu funcionalidad o corrección (`git checkout -b feature/nueva-funcionalidad`).
3. Realiza tus cambios y asegúrate de que el código siga los estándares del proyecto.
4. Haz commit de tus cambios (`git commit -m 'Añade nueva funcionalidad'`).
5. Sube tu rama (`git push origin feature/nueva-funcionalidad`).
6. Abre un Pull Request describiendo tus cambios detalladamente.

## Estándares de Código

- Usa TypeScript para todo el código nuevo.
- Sigue las convenciones de nombres (camelCase para variables, PascalCase para componentes).
- Asegúrate de que los componentes sean reutilizables y sigan la estética de la marca.

## Reporte de Errores

Si encuentras un error, por favor abre un "Issue" usando la plantilla de reporte de errores.
